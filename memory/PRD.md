# My-AlphaAI — Product Requirements Document

## Original Problem Statement
My-AlphaAI is a B2C/SaaS crypto trading signals platform optimized for conversion with AI-powered research, backtesting, strategy management, real-time signals, and admin monitoring.

## Core Architecture
- **Frontend**: React + Tailwind CSS + Shadcn UI + Recharts + Framer Motion
- **Backend**: FastAPI + Motor (Async MongoDB) + WebSockets
- **Integrations**: OpenAI GPT-5.2 (Emergent LLM Key), Stripe, CoinGecko, Resend

### Agent Workers Live Mode (Apr 5, 2026)
- **4 Specialized Agents** running as background workers:
  - **Momentum Scanner** (5min interval): RSI + MACD crossover detection on BTC, ETH, SOL. Generates BUY/SELL signals on divergence/crossover events.
  - **Sentiment Analyzer** (10min interval): Price-action based sentiment scoring. Generates BUY/SELL/HOLD from sentiment score (10-90 scale).
  - **Whale Tracker** (10min interval): Volume spike detection (>1.5x/2x average). Flags large player activity.
  - **Volatility Engine** (10min interval): Bollinger Band compression + ATR breakout. Mean reversion and expansion signals.
- **Agent API**:
  - `GET /api/agents` — All agents with real metrics (accuracy, signal count, status)
  - `GET /api/agents/{id}/signals` — Signals generated by specific agent
  - `PUT /api/agents/{id}/config` — Update agent config (risk_level, timeframe, frequency)
- **Portfolio Endpoints**:
  - `GET /api/portfolio/performance` — Equity curve data
  - `GET /api/portfolio/trades` — Trade history
  - `GET /api/portfolio/summary` — Aggregated portfolio stats
- **All endpoints demo-mode aware**: Real data when DEMO_MODE=false, synthetic when true
- **Frontend AgentsPage**: Rewritten — fetches real agent data, shows LIVE/DEMO badge, View Signals modal with real signals, Configure modal with real parameters
- **Legacy `/api/agents` route** redirects to new system
- Testing: 19/19 backend + all frontend UI passed (iteration 72)
